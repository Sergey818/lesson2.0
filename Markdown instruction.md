# Инструкция для работы с Markdown

## Выделение текста

Чтобы выделить текст курсивом необходимо обрамить его (*). Например *Вот Так*
Чтобы выделить текст полужирным необходимо обрамить его (**) например **Вот так**

## Списки

* список 1
* список 2
1. список 1
2. список 2

## Работа с изображениями
